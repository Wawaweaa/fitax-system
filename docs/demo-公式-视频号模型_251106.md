 以下公式是文件 demo-规则样例_视频号模型_251106.xlsx 中, A列-O列每列的公式，所有公式中引用的列均在Q列-CA列中
 
 - year(年)
      - 公式：=IFERROR(YEAR(U2),"")
      - 精度/单位：INT
      - 备注：A列
  - month(月)
      - 公式：=IFERROR(MONTH(U2),"")
      - 精度/单位：INT
      - 备注：B列
  - order_id(订单号)
      - 公式：=Q2
      - 备注：C列
  - line_count(订单行数)
      - 公式：=SUMPRODUCT(($Q$2:$Q$3064=C2)*1)
      - 备注：D列
  - line_no(订单序位)
      - 公式：=SUMPRODUCT(($Q$2:Q2=C2)*1)
      - 备注：E列
  - internal_sku(平台商品编码/商家编码)
      - 公式：=BG2
      - 备注：F列
  - fin_code(财务核算编码)
      - 公式：=IFERROR(LEFT(F2,FIND("-",F2)-1),F2)
      - 备注：G列
  - qty_sold(销售数量)
      - 公式：=ROUNDUP((BK2+BN2-BT2)/BI2,0)
      - 备注：H列
  - recv_customer(应收客户)
      - 公式：=IF(H2=0,0,BI2*BM2-BT2)
      - 精度：2位
      - 备注：I列
  - recv_platform(应收平台)
      - 公式：=IF(AND(H2>0,E2=1),SUM(AJ2:AM2),-SUM(AJ2:AM2))
      - 精度：2位
      - 备注：J列
  - extra_charge(收价外收费)
      - 公式：=IF(E2=1,AI2,0)
      - 精度：2位
      - 备注：K列
  - fee_platform_comm(扣平台佣金用)
      - 公式：=IF(E2=1,SUM(AS2:AT2),0)
      - 精度：2位
      - 备注：L列
  - fee_affiliate(扣分销佣金)
      - 公式：=IF(E2=1,BA2,0)
      - 精度：2位
      - 备注：M列
  - fee_other(扣其它费用)
      - 公式：=IF(E2=1,SUM(AU2),0)
      - 精度：2位
      - 备注：N列
  - net_received(应到账金额)
      - 公式：I2+J2+K2-L2-M2-N2
      - 精度：2位
      - 备注：O列，以计算值为准，不取任何“源净额”列
